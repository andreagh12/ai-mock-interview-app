# Mission 3 Readme

This project is part of Mission 3 – Turners Car Insurance Project.

Due to the insurance process redesign, management expects that many staff members in the department will be retrained into other roles. The aim of this project was to build a **Mock Job Interview Application using Generative AI** that allows staff members to practice job interviews for another role.

## Built With

- React
- Vite
- Node.js
- Express
- Google Gemini API
- CORS

## How It Works

The application simulates a job interview with an AI interviewer.

The user enters a **job title** and answers interview questions generated by Generative AI. The AI asks a series of questions related to the role and adjusts its responses based on the user's answers.

The interface includes:

- A **Job Title textbox** where users type the role they want to interview for (free-text field)
- A **conversation display area** that shows both the user’s responses and the AI interviewer’s questions
- A **response textbox** where the user types their answer
- A **Submit button** that sends the response to the application and generates the next AI question

The interview begins with the question:

"Tell me about yourself"

The AI then asks **at least six interview questions** related to the role entered by the user.

All questions except the first are generated dynamically using Generative AI and are not hardcoded.

At the end of the interview, the AI interviewer provides feedback on the user’s responses and suggests ways to improve.

## Installation

Install dependencies and run locally:

```bash
npm install
npm run dev
```

The backend runs on:
http://localhost:5050

Run the frontend:

```bash
cd client
npm install
npm run dev
```

The frontend runs on:
http://localhost:5173/

## Features
- AI-generated interview questions
- Dynamic conversation between user and AI interviewer
- Feedback provided after the interview
- Role-specific interview simulation
